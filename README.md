# Login Authentication App

A Java Full Stack Login Authentication Application built using Spring Boot, React (Vite), and PostgreSQL.  
It features JWT-based authentication, password encryption (Spring Security + BCrypt), role-based access control, and audit logging for login and logout tracking.

---

## Features

- User Registration and Login  
- Password Encryption using BCrypt  
- JWT Authentication  
- Role-Based Access (Admin / User)  
- Audit Logging (Login / Logout time tracking)  
- PostgreSQL Database Integration  
- RESTful API Backend + React Frontend  
- Modular structure for scalability  

---

## Tech Stack

| Layer | Technology Used |
|-------|------------------|
| Frontend | React (Vite), Axios, Material UI |
| Backend | Spring Boot (Java 17) |
| Security | Spring Security, JWT |
| Database | PostgreSQL |
| Build Tool | Maven |
| Version Control | Git & GitHub |
